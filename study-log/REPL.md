- Read-Eval-Print-Loop

줄단위 표현식

1. 사용자가 입력한 코드를 읽고
2. 코드를 실행하고
3. 결과를 출력
4. 다시 입력을 기다리는 상태로 돌아간다

일종의 대화형 프로그래밍 환경

- Lisp 언어에서 처음 사용되었음
- 스크립트 언어(Python, Ruby등) 의 전유물 였지만, 최근은 Java 같은 컴파일 언어도 제공해줌

> 쉘 스크립트는 그 자체로 REPL 은 아니다.
> 대화형 쉘 (터미널을 열고 명령어를 한 줄씩 입력하는 환경) 이 REPL
> `sh test.sh` 와 같이 스크립트는 다음 입력을 기다리지 않고 종료
